# Aqua Portfolio

A visually-driven creative portfolio featuring scroll-triggered animations, an infinite image gallery, and immersive interactions.

## Tech Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS v4
- Framer Motion
- Lenis (smooth scroll)
- pnpm monorepo

## Features

- Scroll-driven letter animations
- Infinite parallax image gallery
- Smooth scrolling with gear-tick audio feedback
- Responsive layout

## Getting Started

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```
